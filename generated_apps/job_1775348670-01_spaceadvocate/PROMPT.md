# JacCoder Prompt

Create a web application named SpaceAdvocate aimed at supporting advocacy for NASA funding. The platform should have community forums for users to discuss and collaborate on advocacy efforts, tools for fundraising campaigns, and a petition feature to gather support. Users should be able to organize events to raise awareness about NASA projects. Include a resource library that provides information on ongoing and proposed NASA initiatives. Use Ruby on Rails for the backend, React for the frontend, and SQLite for the database.
