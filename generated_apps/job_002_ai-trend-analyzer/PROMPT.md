# JacCoder Prompt

Build a small full-stack web application called TrendSight that ingests a list of news headlines and groups them into thematic categories. The app exposes a REST API where users can POST a batch of article objects (title, description, source) and receive back a structured response with categories, each containing a name, a one-paragraph summary, and the articles assigned to it. A minimal web dashboard lets users paste raw headlines, trigger the categorization, and browse the grouped results. Categories must be generated by an LLM (GPT-4o-mini). The data model should persist past categorization runs so users can compare how topics shift over time.
